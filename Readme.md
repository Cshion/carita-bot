# Development
## Install dependencies

    cd carita-bot
    npm install

## Set your FB credentials:
### Windows

 	set EMAIL=..@...
 	set PASS=xxxxxx

### Linux

    EMAIL=..@... PASS=xxxxxx node index.js

## Run

    node index
